# Исправление ошибки HTTPS для Telegram Mini App

## Проблема:
```
Please send me a valid URL. https is required.
```

BotFather требует **HTTPS** URL, а вы отправили **HTTP**.

---

## Решение:

### Вариант 1: Использовать домен с HTTPS от Coolify

Coolify обычно автоматически создает HTTPS для ваших доменов. Проверьте:

1. Зайдите в настройки вашего приложения в Coolify
2. Найдите раздел **"Domains"** или **"FQDNs"**
3. Должен быть автоматически создан HTTPS URL, например:
   - `https://q84oskg0cs044ogwkok0os04.91.107.212.137.sslip.io`
   - Или другой домен с SSL

4. Скопируйте **HTTPS** URL (не HTTP!)
5. Отправьте в BotFather

---

### Вариант 2: Настроить свой домен с SSL

Если есть свой домен:

1. В Coolify добавьте ваш домен:
   - Settings → Domains → Add Domain
   - Введите ваш домен (например: `notifications.yourdomain.com`)

2. Coolify автоматически выдаст SSL сертификат через Let's Encrypt

3. После получения сертификата используйте:
   ```
   https://notifications.yourdomain.com
   ```

---

### Вариант 3: Использовать встроенный домен Coolify с HTTPS

1. В настройках приложения найдите **"FQDN"** или **"Public URL"**
2. Должен быть URL вида:
   ```
   https://q84oskg0cs044ogwkok0os04.91.107.212.137.sslip.io
   ```
   (обратите внимание на **https://** в начале!)

3. Скопируйте этот URL
4. Отправьте в BotFather

---

## Проверка:

Перед отправкой в BotFather проверьте в браузере:

1. Откройте URL с **https://** в браузере
2. Должна открыться ваша страница с уведомлениями
3. Если видите ошибку SSL - нужно подождать пока Coolify выдаст сертификат

---

## Частая ошибка:

❌ **НЕПРАВИЛЬНО:**
```
http://q84oskg0cs044ogwkok0os04.91.107.212.137.sslip.io
```

✅ **ПРАВИЛЬНО:**
```
https://q84oskg0cs044ogwkok0os04.91.107.212.137.sslip.io
```

Разница - **http** vs **https**!

---

## Если HTTPS не работает:

1. Подождите 1-2 минуты после деплоя (Coolify выдает SSL автоматически)
2. Проверьте в браузере: откройте `https://your-url` (не http!)
3. Если ошибка SSL - зайдите в настройки домена в Coolify
4. Попробуйте перезапустить приложение

---
